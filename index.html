Hi! Welkom bij mijn custom properties!

Wat is het doel van deze opdracht?

Met deze opdracht kijk ik welke custom properties er wellicht allemaal zijn om ze zo allemaal in te voegen voor mijn eigen huisstijl.

Waarvoor kan je custom properties gebruiken?

  • Voor fonts
  • Voor kleuren
  • Voor borders
  • Voor breedtes en/of hoogtes

    • Welke custom properties heb je verwerkt op deze site?

    Ik heb tijdens dit onderzoek gekeken naar wat allemaal belangrijk is om te implenmenteren. Daarnaast was het ook een kwestie van basis elementen te maken en custom properties daarop in te voegen. Hieronder kan je zien welke code ik heb gebruikt voor de custom properties. Deze heb ik in de html parent gedaan.

    html {
    --primairy-color-orange: #F7A100;
    --primairy-color-white: #FFFFFF;
    --primairy-color-blue: #0071B3;
    --primairy-color-black: #333333;
    --secundairy-color-brown:#53241B;
    --secundairy-color-blue:#478BB3;
    --secundairy-color-yellow:#F8C463;
    --secundairy-color-brown:#66BA96;
    --grayscale-color-lightest-gray:#EDEDED;
    --grayscale-color-light-gray:#CCCCCC;
    --grayscale-color-gray:#666666;
    --grayscale-color-darkest-gray:#333333;
    --content-radius: 3px;
    --regular-text: 16px;
    --fonts:"ProximaNova", sans-serif
    }
